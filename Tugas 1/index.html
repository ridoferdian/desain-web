Struktur HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web. Struktur dasar HTML terdiri dari beberapa elemen utama, yaitu:

  • <!DOCTYPE html>: Deklarasi yang menunjukkan bahwa dokumen ini menggunakan HTML5.
  • <html>: Elemen root yang membungkus seluruh konten halaman web.
  • <head>: Bagian yang berisi informasi meta, judul, dan link ke sumber daya eksternal seperti CSS dan JavaScript.
  • <body>: Bagian yang berisi konten utama halaman web, seperti teks, gambar, dan elemen interaktif lainnya.

Setiap elemen dalam HTML memiliki tag pembuka dan penutup, kecuali untuk elemen kosong seperti <img> atau <br>. Struktur HTML yang baik sangat penting untuk memastikan bahwa halaman web dapat diakses dengan baik oleh pengguna dan mesin pencari.
